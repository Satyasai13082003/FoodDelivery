### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/Satyasai13082003/FoodDelivery.git
    cd FoodDelivery
    ```

2.  **Install Dependencies**
    ```bash
    # For Frontend (if in a client/ directory)
    cd client
    npm install

    # For Backend (if in a server/ directory)
    cd ../server
    npm install
    ```

3.  **Environment Variables**
    *   Create a `.env` file in the root of your backend directory.
    *   Add your environment-specific variables (e.g., Database URL, JWT Secret, API Keys).
    ```
    DB_CONNECTION = your_database_connection_string
    JWT_SECRET = your_super_secret_jwt_key
    ```

4.  **Run the Application**
    ```bash
    # Start the Backend Server (from server/ directory)
    npm start

    # Start the Frontend Development Server (from client/ directory)
    npm start
    ```
